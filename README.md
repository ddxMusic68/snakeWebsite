# snakeWebsite
I made a fullstack website where you can play snake and it comes with a leaderboard

## versions
```
python 3.11.9
Flask 3.1.2
Jinja2 3.1.6
bootstrap 5
```
## installation
```
git clone https://github.com/ddxMusic68/snakeWebsite
cd project
python -u ./server.py
```
## structure
```
src
│--leaderboard.json    -holds leaderboard information
│--server.py    -runs server
└───website
    │--views.py    -main backend
    │--__init__.py    -sets everything up
    ├───static
    │--main.css    -main css
    │--main.js    -main js
    ├───templates
    │--base.html    -base html file
    │--leaderBoard.html    -extends base.html to a leaderboard
    │--snake.html    -extends base.html to the main snake game
```
